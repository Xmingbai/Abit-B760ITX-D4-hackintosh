# 电脑配置

机箱	：闪鳞S300-U3 8升ITX 带前置TYPE-C

主板	：Abit B760ITX D4

CPU	：i5-13400 10核心16线程

显卡	：XFX RX5500XT 4G

延长线：PCIE4.0

内存	：镁光8x2 DDR4 3200 

硬盘	：西数4.0固态SN850 1T

散热 ：利民AXP90-X53

电源	：Tt钢影TRM SFX450 电源

网卡	：兼容AX200/201/210/211 或博通型号
